# irc의 상용 서버

실제 클라이언트와 서버간의 통신이 어떻게 이루어지는지 확인하기 위한 irc 서버와 irssi 클라이언트를 설치하고 실행하는 도커파일입니다.

## 사용법

다음 명령어를 통해 도커 컨테이너 내부로 들어갑니다.
```
docker exec -it irssi bash
```

다음 명령어를 통해 irc 서버가 열려있는 6667포트의 송수신 내역을 확인 할 수 있습니다.
```
tcpflow -i lo port 6667 -c
```
tcpflow가 성공적으로 실행되면 이제 해당 터미널에는 irc 서버에서의 통신 내역을 확인 할 수 있습니다.

이제 직접적으로 클라이언트를 통해 서버에 접속하고 어떤 메시지를 주고받는지 확인 할 수 있습니다.  

다음 명령어를 통해 도커 내부 컨테이너에 접속하고 irssi 클라이언트를 통해 irc 서버에 접속할 수 있습니다.
```
docker exec -it irssi bash
irssi -c 127.0.0.1 -n gyyu
```
